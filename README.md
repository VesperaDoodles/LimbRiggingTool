# Project for The Rookies 2024 
Biped Limb Rig Script with options :

- FK Hand setup
- Reverse Foot Roll Setup
- Bendy Limbs,
- Stretch System
- Automatic Pole Vector Placement

# INSTALLATION
### main.py 
Edit the directory constant variable to be the path to your LimbRiggingTool folder

### In Maya
Load the main.py
